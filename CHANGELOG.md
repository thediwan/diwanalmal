# Changelog

All notable changes to Diwan Al-Mal are documented here.

Format follows [Keep a Changelog](https://keepachangelog.com/en/1.1.0/).
Versioning follows [Semantic Versioning](https://semver.org/) and maps to
`pubspec.yaml` as `versionName+versionCode` (e.g. `1.0.0+1`).

## [1.0.0] - 2026-06-30

### Added

- Initial public release preparation for F-Droid
- Offline-first personal finance management (wallets, transactions, goals, budgets)
- Arabic-first RTL UI with English localization
- Local authentication (PIN, biometrics, recovery code)
- Monthly financial reports with PDF export
- Scheduled local backup (`.dmbackup` archives)
- Multi-currency support with frozen exchange rates
- GPL-3.0-or-later license
